ABOUT THE GAME

It is a game to remember the sequence of the buttons that blinks one by one after you start the game. With each level one extra button blinks and you have to click the sequence from the start to reach the next level.

HOW TO PLAY

  • First, Press the START button to start the game.
  • A button will blink with a sound along with the blink. Let it be "Green".
  • Now click the "Green"button and now you have reached the next level of the game.
  • Again a button will blink along with a sound.Let it be "Red".Now you have to click the sequence that is the "Green" button first and then the "Red" button.
  • If you clicked both the buttons the level will change to 3 and again the other button will blink.Let it be "Green" again.
  • Now 3 buttons need to be pressed in sequence("Green" -> "Red" -> "Green") to reach the next level.
  • In this way , you can test the remembering skills of your mind by completing the sequence again and again.
